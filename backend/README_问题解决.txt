问题1：Maven未安装 ❌
-------------------
看到错误："Maven未安装或未配置环境变量"

解决方法（3选1）：

【方法1：安装Maven】⭐⭐⭐
-----------------------
1. 查看：安装Maven指南.md
2. 下载安装Maven
3. 配置环境变量
4. 重启命令行
5. 重新运行start.bat

【方法2：使用IDEA】⭐⭐⭐（最简单）
----------------------------------
1. 打开IntelliJ IDEA
2. File -> Open -> 选择backend目录
3. 等待依赖下载
4. 找到StarWarApplication.java
5. 右键 -> Run

不需要安装Maven，IDEA自带！

【方法3：排查】⭐⭐
------------------
1. 查看：Maven问题排查.txt
2. 按照指南操作

---------------------------------------

问题2：数据库连接失败 ❌
---------------------
看到错误：Access denied / Cannot connect

解决方法：
1. 检查MySQL服务是否启动
   - 打开"服务"管理器
   - 找到MySQL服务
   - 确保状态为"正在运行"

2. 检查密码是否正确
   - 打开：src/main/resources/application.yml
   - 第8行：password: 你的MySQL密码

3. 测试连接
   - 用MySQL Workbench连接数据库
   - 确认用户名密码正确

---------------------------------------

问题3：端口被占用 ❌
------------------
看到错误：Port 8080 is already in use

解决方法：
1. 关闭占用8080端口的程序
2. 或修改端口：
   - 打开：src/main/resources/application.yml
   - 修改：port: 8081

---------------------------------------

问题4：依赖下载失败 ❌
-------------------
看到错误：Could not resolve dependencies

解决方法：
1. 配置Maven国内镜像
2. 编辑：%MAVEN_HOME%\conf\settings.xml
3. 添加阿里云镜像（见安装Maven指南.md）

---------------------------------------

问题5：找不到配置 ❌
------------------
看到错误：找不到配置文件 application.yml

解决方法：
1. 确认您在backend目录下运行
2. 确认配置文件存在
3. 文件路径：backend/src/main/resources/application.yml
