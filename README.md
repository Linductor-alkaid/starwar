# 🚀 星际防卫战

一个基于HTML5 Canvas的太空射击游戏，使用纯前端技术开发。

## 🎮 游戏特色

- 流畅的飞船控制和射击机制
- 多种敌人类型和AI行为
- 道具系统（生命值、弹药、分数加成）
- 渐进式难度增加
- 炫酷的视觉效果和音效

## 🕹️ 操作方式

- **← →** 方向键：移动飞船
- **空格键**：发射激光（消耗弹药）
- 躲避敌人攻击，收集道具提升能力！

## 🎁 道具说明

- 🟢 **绿十字**：恢复生命值
- 🔵 **蓝菱形**：补充弹药  
- 🟡 **黄星星**：额外得分

## 🛠️ 技术栈

### 前端
- HTML5 Canvas
- JavaScript (ES6+)
- CSS3 动画效果
- 响应式设计

### 后端 ✅
- Spring Boot 2.7.18
- MySQL 8.0
- MyBatis
- JWT 认证
- Swagger文档
