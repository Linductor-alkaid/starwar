## ✅ 任务列表

### 任务1：环境搭建 - 安装JDK 8
- **优先级**：🔴 高
- **预计时间**：1小时
- **负责人**：全体成员
- **状态**：⏳ 待开始

#### 具体步骤
1. [ ] 访问Oracle或OpenJDK官网下载JDK 8
2. [ ] 安装JDK到本地
3. [ ] 配置环境变量 `JAVA_HOME`
4. [ ] 将 `%JAVA_HOME%\bin` 添加到系统PATH

### 任务6：配置项目并执行数据库脚本
- **优先级**：🔴 高
- **预计时间**：1小时
- **负责人**：后端开发人员 + 数据库设计人员
- **状态**：⏳ 待开始

#### 具体步骤
1. [ ] 创建 `src/main/resources/application.yml`
2. [ ] 配置数据库连接信息
3. [ ] 在MySQL Workbench中打开 `docs/database/init.sql`
4. [ ] 执行数据库初始化脚本
5. [ ] 验证表是否创建成功

#### application.yml配置示例
```yaml
server:
  port: 8080

spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/star_war_db?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的密码

mybatis:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: com.starwar.entity
  configuration:
    map-underscore-to-camel-case: true
```

#### 验收标准
- ✅ 配置文件正确无误
- ✅ 数据库 `star_war_db` 创建成功
- ✅ 4张表创建成功（user, game_record, post, reply）
- ✅ 测试数据插入成功
- ✅ IDEA中能够连接到数据库
