本指南将帮助您在30分钟内完成开发环境的搭建，并运行一个简单的后端服务。

## 🚀 快速开始（5个步骤）

### 第1步：安装JDK 8
1. 下载JDK 8：https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
2. 安装JDK
3. 配置环境变量：
   - Windows: 系统属性 -> 环境变量 -> 新建 `JAVA_HOME` -> 值为JDK安装路径
   - 将 `%JAVA_HOME%\bin` 添加到 `Path` 环境变量
4. 验证：打开命令行，输入 `java -version`

### 第2步：安装MySQL 8.0
1. 下载MySQL：https://dev.mysql.com/downloads/mysql/
2. 安装MySQL（记住root密码）
3. 使用MySQL Workbench执行初始化脚本：
   ```sql
   -- 打开 docs/database/init.sql
   -- 在Workbench中执行整个脚本
   ```
4. 验证：在Workbench中能看到 `star_war_db` 数据库

### 第3步：安装Maven
1. IDEA自带Maven，通常无需额外安装
2. 如需独立安装：https://maven.apache.org/download.cgi
3. 配置国内镜像（可选，加速下载）：
   编辑 `maven/conf/settings.xml`：
   ```xml
   <mirror>
     <id>aliyun</id>
     <mirrorOf>central</mirrorOf>
     <name>阿里云Maven镜像</name>
     <url>https://maven.aliyun.com/repository/public</url>
   </mirror>
   ```

### 第4步：创建Spring Boot项目
1. 打开IntelliJ IDEA
2. File -> New -> Project
3. 选择 Spring Initializr
4. 配置信息：
   - Group: `com.starwar`
   - Artifact: `star-war-backend`
   - Type: Maven Project
   - Language: Java
   - Java Version: 8
5. 点击 Next
6. 选择依赖：
   - ☑ Spring Web
   - ☑ MyBatis Framework
   - ☑ MySQL Driver
   - ☑ Lombok
7. 点击 Finish

### 第5步：配置并运行
1. 修改 `src/main/resources/application.properties` 或创建 `application.yml`：
   ```yaml
   server:
     port: 8080
   
   spring:
     datasource:
       driver-class-name: com.mysql.cj.jdbc.Driver
       url: jdbc:mysql://localhost:3306/star_war_db?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
       username: root
       password: 你的MySQL密码
   ```
2. 运行主类 `StarWarApplication`
3. 看到 "Started StarWarApplication" 表示启动成功！

## ⚠️ 常见错误解决

### 错误1：端口被占用
```
Error: Port 8080 is already in use
```
**解决**：修改 `application.yml` 中的 `server.port` 为其他端口（如8081）

### 错误2：数据库连接失败
```
Access denied for user 'root'@'localhost'
```
**解决**：检查MySQL的用户名和密码是否正确

### 错误3：找不到类
```
ClassNotFoundException
```
**解决**：刷新Maven项目（IDEA右侧Maven面板 -> Reload All Maven Projects）

### 错误4：编译错误
```
找不到符号
```
**解决**：确保安装了Lombok插件（IDEA: Settings -> Plugins -> 搜索Lombok -> Install）
