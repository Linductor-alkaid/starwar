## 🛠️ 项目技术栈

### 后端
- **框架**：Spring Boot 2.x
- **语言**：Java 8
- **数据库**：MySQL 8.0
- **ORM**：MyBatis
- **认证**：JWT
- **构建工具**：Maven

### 前端（待集成）
- **框架**：Vue 2.x
- **UI组件**：Element UI
- **HTTP客户端**：Axios

### 开发工具
- **IDE**：IntelliJ IDEA
- **数据库管理**：MySQL Workbench
- **API测试**：Postman
- **版本控制**：Git
